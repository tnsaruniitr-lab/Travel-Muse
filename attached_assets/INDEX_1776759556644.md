## How this library works

Each blog post lives in its own subfolder named after the slug. Inside each folder:

- **`[slug].html`** — the complete self-contained HTML page (all schemas + base64 images, openable in browser)
- **`[slug]-schemas.json`** — all JSON-LD schemas as a clean array
- **`[slug]-head-meta-and-schemas.html`** — `<head>` snippet for CMS pasting
- **`[slug]-body.html`** — `<article>` body snippet for CMS pasting
- **`README.md`** — what's in the bundle, pre-publish checklist, image notes

## Adding a new blog

When you ask Claude to generate a new TRYPS blog, it should drop the bundle into a new subfolder here named after the slug. Append a row to the table above.
